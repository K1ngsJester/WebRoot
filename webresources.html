Web Resources:

Site: Rating:
W3Schools.com: is a great place for anyone to start. It is a great resource for looking up quick HTML, CSS, Javascript and much more. This site has tutorials in most code languages and examples that can be edited by users.
CodeAcademy.com: is another great area to learn at your own pace. Unlike w3schools, you have to sign up (for free) to access some courses. Taking courses here will help you learn how to be a web developer in as long as it takes to learn the material.
StackOverFlow.com: This site is a great place to ask questions and get intermediate to advanced help in your solutions. I have accessed this site more than a couple times on specific questions and gotten answers that at times can be diffcult to understand but work great.
Lynda.com: Also is now LinkedIn Learning is great for professional subjects being taught online. With a school email you can access some of these classes to learn and build on the side. Also you can put these credentials on your LinkIn profile along with any degrees for the field.
CodeWars.com: This site is used to sharpen and creatively solve problems created by others or the site. This is a great place to share and build experiances with others for free. One downside is you need to have some coding experance since it is not for complete beginners.
